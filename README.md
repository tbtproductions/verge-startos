# Verge for StartOS

Net worth tracker with live Bitcoin, crypto, and stock prices running on your own hardware.

## Sideload Instructions

1. Download the latest verge_x86_64.s9pk from the Releases page
2. Open your StartOS dashboard in a browser
3. Click the upload icon in the top toolbar
4. Select the s9pk file
5. Wait for installation to complete
6. Click Start on the Verge service

## First Run

- Bitcoin and crypto prices work immediately with no setup
- For stock prices go to Actions then Set Finnhub API Key
- Get a free API key at finnhub.io
